Embed GitHub Contributions
Embed your GitHub contributions graph on any page with a simple configurable URL.

Getting Started

1. Embed Iframe

<iframe class="github-embed" src="https://contributions.okcode.club/embed.html?username={github username}&axis=false&dark=true" frameborder="0"></iframe>

1. Style accordingly

.github-embed{
  display: block;
  max-width: 1000px;
  height: 109px;
  width: 100%;
  margin-left: auto;
  margin-right: auto;
}

Embed URL Parameters

Key Default Description
username yyx990803 GitHub username for whom the contribution graph is to be retrieved.
axis true Indicates whether or not to include graph axis labels.
dark false Enables dark mode styling if set to true and assumes background will be dark coloured.
